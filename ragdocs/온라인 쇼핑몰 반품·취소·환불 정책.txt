온라인 쇼핑몰 반품·취소·환불 정책

1. 반품 방법
반품 절차

마이페이지 또는 주문내역에서 교환/반품 신청

택배기사 또는 지정된 배송 방식으로 상품 회수

반품 상품 검수 완료 후 환불 처리

반품 기한

단순 변심

자체 배송 상품: 상품 수령 후 30일 이내

판매자 배송 상품: 상품 수령 후 7일 이내

상품 불량 또는 오배송

상품 수령 후 3개월 이내

또는 불량/오배송 사실을 인지한 날로부터 30일 이내

2. 반품 비용 부담 기준

※ 반품 사유에 따라 고객 또는 판매자가 비용을 부담할 수 있으며,
발생 비용 및 환불 예정 금액은 반품 신청 시 확인 가능합니다.

단순 변심의 경우
자체 배송 상품

멤버십 혜택 대상: 무료 반품 또는 교환 가능

그 외의 경우:

(총 주문금액 - 반품 상품금액)이 기준 금액 미만인 경우: 반품비 상향 부과

기준 금액 이상인 경우: 반품비 하향 부과

판매자 배송 상품

최초 배송비를 포함한 왕복 배송비 발생 가능

무료 배송 상품도 반품 시 반품 배송비는 고객 부담

도서산간 지역 또는 설치 상품의 경우 추가 비용 발생 가능

상품 불량 또는 오배송

반품 비용 발생하지 않음

3. 주문 취소 방법
배송 상품

결제 완료 또는 상품 준비 중 상태에서는 주문 취소 가능

상품이 이미 발송된 경우 취소 불가, 반품 절차로 진행

티켓·이용권 상품 (공연, 여행, 문화 등)

주문내역에서 취소 신청

취소 신청서 작성

사용 여부 확인 후 처리

미사용: 환불 진행

사용 이력 존재: 취소 불가 또는 수수료 차감 환불

※ 상품에 따라 취소 수수료가 부과될 수 있음

4. 환불 시점
결제 수단별 환불 처리 기준

간편결제 머니/포인트

환불 승인 후 당일 환불

적립금/캐시

취소 완료 후 즉시 재적립

신용카드 / 체크카드

평일 기준 3~7일 이내 승인 취소

휴대폰 결제

당월 취소: 당월 요금 미청구 또는 한도 복구

부분 취소 또는 익월 취소: 등록된 환불 계좌로 환불

무통장 입금

등록된 환불 계좌 기준 1~2영업일 이내 환불

환불 계좌 미등록 시 간편결제 머니로 환불될 수 있음

5. 취소·반품 불가 사유
공통 불가 사유

단순 변심에 의한 취소/반품 기한 초과

주문 제작 또는 제작이 이미 진행된 상품

고객 부주의로 인한 파손, 고장, 오염

상품 사용 또는 소비로 가치가 현저히 감소한 경우

화면 해상도 차이로 인한 색상 또는 이미지 차이

제조사 정책 변경 또는 가격 변동에 따른 반품 및 보상 요구

상품 상세 페이지에 별도 고지된 제한 사유

6. 상품군별 취소·반품 제한 예시
상품군	취소·반품 제한 사유
의류 / 잡화	택(TAG) 제거, 라벨 훼손, 세탁 등으로 상품 가치 감소
식품 / 화장품	사용, 시간 경과, 일부 소비로 가치 감소
가전 / 설치 상품	설치 또는 사용 후 단순 변심
자동차 용품	개봉 또는 장착 후 단순 변심
CD / DVD / 게임 / 도서	복제 가능 상품의 포장 훼손
시리얼 코드 상품	시리얼 번호 노출 또는 등록
노트북 / 데스크탑	소프트웨어 분리·삭제로 재판매 불가
